# TalKnot（トークノット）アプリ概要・使い方

社内向け 営業商談・ロープレ評価 Web アプリ。
Talk（話す）＋ Knot（結び目・絆）。動画/音声からAIが商談を振り返り、
お客様の感情の動きをタイムスタンプ付きで可視化する **ポジティブな振り返りツール** です。

---

## 1. これは何のアプリ？

- 住宅営業の商談・ロープレ動画（または音声）をアップロードすると、
  AI（Google Gemini）が **声のトーン・間・発話比率** まで読み取って評価します。
- 「どこが良かったか」「ここでお客様はこう感じていた」「こう言えたらもっと刺さる」を
  **Before → After のタイムスタンプ付き**で示します。
- 批判ではなく、次の商談が良くなるための **前向きな振り返り** が目的です。

### アクセス
- **URL: https://talknot-lts.streamlit.app**
- スマートフォン・PC どちらからも利用可（ブラウザのみ。インストール不要）。
- **ログインできるのは `@life-time-support.com` のGoogleアカウントのみ**（ドメイン制限）。

## 2. 評価のしくみ（5項目 × 2軸）

5つの観点それぞれを、**2つの視点**で 1〜5 点で採点します。

### 評価5項目
| | 項目 | 見るポイント |
|---|---|---|
| ① 🌱 | 追加検討が増えたか | 検討項目・興味が広がり、次の一歩につながったか |
| ② 🤸 | 臨機応変に対応ができるか | 想定外の質問に柔軟に切り返せたか |
| ③ 💗 | お客様の感情をキャッチできているか | トーン・間・表情から感情を拾えたか |
| ④ 🔍 | お客様の背景を深掘れたか | 実家との距離・子供・引越し時期など秘密領域まで踏み込めたか |
| ⑤ ✨ | 雰囲気よくワクワクしたか | 「任せたら理想の家ができる」期待が高まったか |

### 評価2軸
| 軸 | 内容 |
|---|---|
| 🎯 模範トーク視点 | 登録された模範トーク（無ければ業界トップの理想トーク）と比べてどれだけ近いか |
| 💼 営業プロ視点 | 住宅営業のトップセールス兼コーチとして見た完成度・次の一手 |

→ 結果画面では各項目に **🎯模範 X/5 ｜ 💼営業 Y/5** の2スコアと講評が並びます。

## 3. 使い方（3ステップ）

1. **ログイン** — URLを開き「Googleでログイン」。会社アカウントを選ぶだけ。
2. **動画を選ぶ** — 「商談を評価する」で
   - 手元の動画/音声ファイルをアップロード、または
   - Google Drive（Meet録画）から選択。
3. **評価を見る** — 数十秒〜数分でAI評価が表示。総合スコア・項目別2軸スコア・
   全体講評・シーン別 Before→After が出ます。履歴も自動保存されます。

### 管理者だけができること
- **模範トークの登録**（「⭐模範トーク」タブ）。お手本トークを登録すると
  🎯模範トーク視点の採点基準が、その実際の模範に切り替わります。
- 現在のアプリ管理者：`kkyoya@` / `hkumada@`（社内設定で増減可）。

## 5. よくある質問

**Q. データはどこに保存される？**
A. 評価結果は社内運用のストレージに保存。動画自体はAI解析のために一時的に送られます。

**Q. 誰でも見られる？**
A. いいえ。`@life-time-support.com` のGoogleログインが必須で、他ドメインは弾かれます。

**Q. AIの評価は厳しい？**
A. 常にポジティブ設計です。良い点を具体的に褒めたうえで「次の一手」を示します。
