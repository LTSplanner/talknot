ryouchiku@life-time-support.com 様

社内の営業ロープレ評価ツール「TalKnot」で、各営業メンバーの Google Meet 録画を
（本人ログインなしで）読み取るための設定をお願いしたいです。**読み取り専用**で、
書き込み・削除はできない設定です。

対象メンバー（8名）の録画を評価に使います：
- s.kageyama@ / hkumada@（模範）/ kkyoya@（模範）/ yhoshino@ / ynakatani@ / amoritani@ / toshima@ / manguri@

---

## お願いしたい作業（2つ）

### ① GCP：サービスアカウントと鍵の作成
1. Google Cloud Console の対象プロジェクトで **Google Drive API** を有効化
2. 「IAMと管理」→「サービスアカウント」→ 作成（例：`talknot-drive`）
3. 作成したアカウントの「キー」→ JSON 鍵を発行（このファイルは運用者へ**安全な方法で**共有してください。メール本文やチャットへの貼り付けは避けてください）
4. そのサービスアカウントの「**クライアント ID（数字の長いID）**」を控える

### ② Admin Console：ドメイン全体の委任（超管理者のみ）
1. 管理コンソール →「セキュリティ」→「アクセスとデータ管理」→「**API の制御**」
2. 「**ドメイン全体の委任**」→「新しく追加」
3. 入力：
   - **クライアント ID**：①-4 で控えた数字のID
   - **OAuth スコープ**：
     ```
     https://www.googleapis.com/auth/drive.readonly
     ```
4. 承認して保存

## 完了後に運用者へ共有いただきたいもの
- ① で発行した **JSON 鍵ファイル**（安全な経路で）
- 設定した **クライアント ID**（確認用）
